== Description ==
This plugin allows users to upload and set a custom image for the WordPress login dashboard. It's an easy and quick way to personalize your login page without needing any code.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory or install the plugin directly from the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings > Login Image to upload and configure your custom image.
